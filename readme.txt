TalentLMS Integration with WordPress for Single Sign-On (SSO) login for users via WordPress, and TalentLMS user sync from WordPress to TalentLMS.

== Description ==

The Single sign on into TalentLMS User Sync Integration plugin enables integration of WordPress sites with TalentLMS. It allows configuration of Single Sign-On(SSO) between WordPress and TalentLMS for seamless login into TalentLMS using WordPress. 

This TalentLMS sync solution is a one-stop shop for user management on both your TalentLMS platform and your WordPress site. From the WordPress site, users may be created and modified directly in TalentLMS.

This WordPress SAML IDP SSO solution provides SAML SSO capability to your WordPress site, converting it to a SAML compliant Identity Provider which can be configured with any SAML compliant Service Provider.


== Key Features ==

* <strong>Single Sign-On (SSO)</strong>
Using your WordPress user store as the identity provider for authenticating your users with WordPress, enable safe one-click login into TalentLMS using SAML Single Sign-On (SSO) (WP as SAML IDP).

* <strong>Easy User synchronization with TalentLMS</strong>
Easily sync user data from WordPress (WP) to TalentLMS and create and update the learners profile fields in the TalentLMS from WordPress on a single click.
